# my-nn-zero-to-hero

Karpathy「Neural Networks: Zero to Hero」code-along —— 我自己一行行敲的版本(AI 全栈学习 阶段二:LLM 专精)。

## 环境
- venv:`~/.venvs/ml312/bin/python`(Python 3.12 + numpy + matplotlib + graphviz + torch)
- PyCharm:把本工程解释器指到上面这个 venv
- 画 loss 曲线时用 `%matplotlib inline`(ipympl/widget 在 PyCharm 里不稳)

## 怎么用
跟着视频**自己敲**,卡住了再对照官方完整 notebook:`~/code/nn-zero-to-hero/lectures/`
